Overview

StreamHash provides native, high-performance implementations of fast non-cryptographic hash algorithms for .NET 10+. Process data incrementally with streaming APIsâ€”perfect for large files, network streams, or real-time data processing.

Version 1.3.0 includes 21 native algorithm implementations with a unified HashFacade API that automatically selects the best implementation available. Algorithms that aren't natively implemented will use BouncyCastle or System.Security.Cryptography fallbacks.

Quick Start

$ dotnet add package StreamHash

Unified HashFacade API

using StreamHash.Core;

// One-shot hashing - returns byte array
byte[] hash = HashFacade.Hash(HashAlgorithm.MurmurHash3_128, data);

// One-shot to hex string
string hex = HashFacade.HashHex(HashAlgorithm.CityHash64, data);

// Streaming for large files
using var stream = File.OpenRead("large-file.bin");
byte[] hash = HashFacade.Hash(HashAlgorithm.SpookyHashV2, stream);

// Create reusable streaming instance
var hasher = HashFacade.CreateStreaming(HashAlgorithm.SipHash_2_4);
hasher.Update(chunk1);
hasher.Update(chunk2);
byte[] result = hasher.Finalize();

Keyed Hashing

// SipHash with custom key (16 bytes)
byte[] key = new byte[16] { /* your key */ };
byte[] hash = HashFacade.HashKeyed(HashAlgorithm.SipHash_2_4, data, key);

// HighwayHash with key (32 bytes)
byte[] highwayKey = new byte[32] { /* your key */ };
byte[] hash = HashFacade.HashKeyed(HashAlgorithm.HighwayHash64, data, highwayKey);
