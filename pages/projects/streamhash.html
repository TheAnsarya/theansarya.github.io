Overview

StreamHash provides native, high-performance streaming implementations of popular hash algorithms for .NET 10+. Process data incrementally with streaming APIs‚Äîperfect for large files, network streams, or real-time data processing.

Version 1.8.0 includes 16 native streaming implementations and a unified HashFacade API supporting all 70 algorithms. Hash multi-gigabyte files with minimal memory footprint (~16MB regardless of file size). Algorithms without native implementations use BouncyCastle or System.Security.Cryptography fallbacks.

Quick Start

$ dotnet add package StreamHash --version 1.8.0

Unified HashFacade API

using StreamHash.Core;

// One-shot hashing - returns byte array
byte[] hash = HashFacade.ComputeHash(HashAlgorithm.MurmurHash3_128, data);

// One-shot to hex string
string hex = HashFacade.ComputeHashHex(HashAlgorithm.CityHash64, data);

// Streaming for large files
using var stream = File.OpenRead("large-file.bin");
byte[] hash = HashFacade.ComputeHash(HashAlgorithm.SpookyHashV2, stream);

// Create reusable streaming instance
using var hasher = HashFacade.CreateStreaming(HashAlgorithm.SipHash_2_4);
hasher.Update(chunk1);
hasher.Update(chunk2);
byte[] result = hasher.FinalizeBytes();

Batch Streaming (All 70 Algorithms)

// Hash a file with ALL 70 algorithms in parallel
using var multi = HashFacade.CreateAllStreaming();

using var stream = File.OpenRead("file.bin");
byte[] buffer = new byte[16 * 1024 * 1024]; // 16MB buffer
int read;
while ((read = stream.Read(buffer)) > 0) {
    multi.Update(buffer.AsSpan(0, read));
}

// Get all 70 hashes as hex strings
Dictionary<string, string> results = multi.FinalizeAll();
// results["SHA256"] = "abc123..."
// results["MurmurHash3_128"] = "def456..."

Keyed Hashing

// SipHash with custom key (16 bytes)
byte[] key = new byte[16] { /* your key */ };
byte[] hash = HashFacade.ComputeHashKeyed(HashAlgorithm.SipHash_2_4, data, key);

// HighwayHash with key (32 bytes)
byte[] highwayKey = new byte[32] { /* your key */ };
byte[] hash = HashFacade.ComputeHashKeyed(HashAlgorithm.HighwayHash64, data, highwayKey);
