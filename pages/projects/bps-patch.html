Overview

BPS Patch is a modern .NET 10 implementation of the BPS (Binary Patch System) format‚Äîthe standard way to distribute ROM hacks. Create patches that transform an original ROM into a modified version without distributing copyrighted content.

This library focuses on performance: SA-IS suffix arrays for optimal match detection, ArrayPool memory management to reduce GC pressure, and multiple algorithm choices for different file sizes.

Library Usage

using BpsPatch.Core;

// Create a patch
BpsEncoder.CreatePatch(
	new FileInfo("original.sfc"),
	new FileInfo("modified.sfc"),
	new FileInfo("patch.bps"),
	"My ROM Hack v1.0");

// Apply a patch
var result = BpsDecoder.ApplyPatch(
	new FileInfo("original.sfc"),
	new FileInfo("patch.bps"),
	new FileInfo("output.sfc"));
