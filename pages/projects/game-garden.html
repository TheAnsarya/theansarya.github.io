Overview

Game Garden is a collection of fully disassembled and documented retro games. Each game proves the bidirectional ğŸŒ· Flower Toolchain works end-to-end: disassemble with Peony, store metadata in Pansy, reassemble with Poppy, and get a byte-identical ROM.

Beyond proving the toolchain, Game Garden serves as a reference for ROM hackers, researchers, and preservationists. Every game includes extracted assets, comprehensive documentation, and working build scripts.
