Overview

Subrom is a modern ROM collection manager with a web-based interface. Think RomVault or ClrMame Pro, but with a React UI, real-time progress updates via SignalR, and support for offline drives and network storage.

Verify your collection against No-Intro, TOSEC, or GoodTools DAT files. See exactly which ROMs you have, which are missing, and which need to be renamed or moved.
