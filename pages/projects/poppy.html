Overview

Poppy is a multi-system assembly compiler that targets classic gaming platforms including NES (6502), SNES (65816), and Game Boy. It's the "assembly" half of the ğŸŒ· Flower Toolchain, designed to work hand-in-hand with Peony (disassembler) and Pansy (metadata format).

The goal is simple: one assembler for all retro platforms. Write your code in .pasm files, and Poppy compiles them into working ROMs that run on real hardware or emulators.

Quick Start

; hello.pasm - NES Hello World
.platform nes
.org $8000

reset:
	sei
	cld
	ldx #$ff
	txs
	; ... initialization code

.org $fffa
.word nmi_handler
.word reset
.word irq_handler
$ poppy build hello.pasm --output hello.nes
